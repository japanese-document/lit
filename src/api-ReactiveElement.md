# ReactiveElement

## class ReactiveElement

要素のプロパティや属性を取り扱う要素のベースクラスです。
プロパティが変更されると、updateメソッドが非同期で実行されます。
更新のレンダリング処理を変更したい場合はupdateメソッドをサブクラスで実装します。

### Import

```
import { ReactiveElement } from 'lit';
```
